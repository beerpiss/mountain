# mountain
A [discord.ts](https://github.com/oceanroleplay/discord.ts) bot written for a club.

## Setting up
Install dependencies:
```bash
# Omit --no-optional to install optional for speed improvments.
npm install --production [--no-optional]

# If you're going to work on Mountain, install everything:
npm install
```

Fill `src/setup.ts` with appropriate values, then run `npm run config` and `npm run scrape`.

Finally, you can build the bot with `npm run build` and serve it with `npm run serve`.
